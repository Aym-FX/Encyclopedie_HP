# Encyclopedie_HP
Sujet de Projet Vue.js : Encyclopédie Interactive de l'Univers de Harry Potter

## Description
Cette application permet à n'importe quel utilisateur de consulter les données liée au domaine de la saga Harry Potter. 
L'utilisateur peut consulter les différentes potions, sorts, personnages et livres Harry Potter, sans avoir besoin de s'inscrire ou se connecter.
Il peut s'il le souhaite rechercher quelque chose.

## Guide d'intallation
Pour installer cette application il suffit sur un terminal, dans le fichier Encyclopédie_HP, de saisir "npm install" pour que toutes les extensions s'installent.

## Guide d'utilisation
Pour utiliser cette documentation Harry Potter. Il suffit de la lancer avec "npm run dev" dans le fichier src du projet.
Une fois lancer, il suffit de cliquer sur le lien du localhost et l'utilisateur sera rediriger sur la page d'accueil de la documentation.
Sur la page d'accueil il y a 4 boutons permettant d'aller sur la page des potions, des sorts, des personnages et des livres.
Une fois sur l'une de ces pages, l'utilisateur peut revenir à la page d'accueil, faire une recherche s'il souhaite avoir directement ce qu'il cherche, 
ou alors défiler la page (qui est composé de 25 éléments). En bas de page il y a 4 boutons pour aller à la première page, aller à la page précédente,
aller à la page suivante et aller à la dernière page (la pagination n'est pas dans la section livres puisqu'il n'y en a que 7). La page 
actuelle est affiché ainsi que le nombre de pages totales, l'utilisateur peut aussi saisir la page qu'il souhaite.
